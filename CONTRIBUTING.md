# Contributing to Kairos

Kairos is in active pre-release development. We are not yet accepting external
contributions, but we will be soon.

## How You Can Help Right Now

- **Try it out** and file issues for bugs or usability problems
- **Star the repo** to help with discoverability
- **Share feedback** via GitHub Issues

## When We Open Contributions

We plan to open contributions after the v0.1.0 stable release. At that point
this file will be updated with development setup, coding standards, and PR guidelines.

## Code of Conduct

We are committed to providing a welcoming experience. Please be respectful
in all interactions.
